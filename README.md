# NAORU整体 渋谷店 リスティング広告用LP プロジェクト

リスティング広告（Google広告・Yahoo!広告）からの流入を前提に、CVR（コンバージョン率）最大化を目的としたランディングページの戦略・構成・実装を行うプロジェクトです。

## 最終ゴール
**初回体験予約（3,500円／通常6,600円）の申込み獲得**

## サービス前提
| 項目 | 内容 |
|---|---|
| 出店エリア | 東京・渋谷 |
| 通常料金 | 1回30分 6,600円 |
| 回数券 | 4回 25,000円（1回あたり6,250円） |
| 初回限定価格 | 3,500円（通常6,600円） |
| ターゲット年齢 | 30〜50代 男女 |
| 強み候補 | 慢性痛／肩こり／腰痛／姿勢改善 |

## このプロジェクトの進め方

1. **Phase 1（本フェーズ）**：戦略・構成・コピーの策定（本ドキュメント群）
2. **Phase 2**：HTML/CSS/JS によるLP実装
3. **Phase 3**：計測タグ設置（GA4 / Google広告 / Yahoo!広告 / LINEタグ）と初期広告入稿
4. **Phase 4**：A/Bテストと改善
