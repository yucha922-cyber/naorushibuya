# 04. リスティング広告との連動

> LPと連動させるべき広告文・キーワード・メッセージマッチを定義します。
> Google広告・Yahoo!広告のいずれも、本ドキュメントに準拠した運用を行ってください。

---

## 4-1. 全体戦略

### 設計思想
- **広告KW** ＝ 検索ボリュームの大きい症状KW（肩こり・腰痛・整体）でリーチを最大化
- **広告文** ＝ FVキャッチと完全に同じ「姿勢×AI×初回3,500円」のメッセージで設計
- **LP** ＝ 流入元のキーワードによって出し分け可能な設計（メッセージマッチ最適化）

### キャンペーン構成（推奨）

| キャンペーン | 主要KW | 想定LP |
|---|---|---|
| **C1：症状KWキャンペーン** | 肩こり 渋谷／腰痛 渋谷／首こり 渋谷 など | メインLP（共感パートで該当症状を冒頭表示） |
| **C2：エリアKWキャンペーン** | 整体 渋谷／渋谷 整体 駅近／整体院 渋谷 など | メインLP |
| **C3：悩み・目的KWキャンペーン** | 姿勢矯正 渋谷／猫背 渋谷／デスクワーク 整体 など | メインLP（メイン訴求が姿勢のため最も親和性高） |
| **C4：競合・指名KWキャンペーン** | カラダファクトリー 渋谷／NAORU整体 渋谷 など | メインLP |
| **C5：リマーケティング** | サイト来訪者・予約未完了者 | メインLP＋限定オファー強調版 |

## 4-4. 広告文のNG表現（再確認）

[03-compliance-and-beforeafter.md](03-compliance-and-beforeafter.md) で定義したNG表現は、広告文でも一切使用しません。特に広告は文字数が短いため、つい強い表現に走りがちですが、以下は厳禁です：

- ❌ 「肩こりが治る」
- ❌ 「腰痛が改善」
- ❌ 「効果保証」
- ❌ 「渋谷No.1」
- ❌ 「完治」「根治」
- ❌ 「医学的根拠」
- ❌ 「絶対」「必ず」

特に**Google広告・Yahoo!広告自体のポリシー**でも「医薬品的効能効果の標榜」「客観的根拠のない最上級表現」は審査で停止される可能性が高いです。

## 4-5. メッセージマッチの設計

### メッセージマッチとは
広告クリック→LP着地時に「広告で見た内容と、LPの第一印象が一致している」状態。
ずれると **直帰率↑・CVR↓** の典型的失敗パターン。

### 整合ルール

| 広告で謳ったこと | LPで必ず再現すべき場所 |
|---|---|
| 「初回3,500円」 | FV内のバッジ・サブコピー・CTAボタン文言 |
| 「AI姿勢分析」 | FVキャッチ or FV直下のセクション |
| 「渋谷駅徒歩2分」 | FVバッジ・店舗情報 |
| 「国家資格者在籍」 | FVバッジ・施術者紹介 |
| 「ボキボキしない」 | NAORUの独自アプローチ理由②・FAQ |
| 「強引な勧誘なし」 | 施術の流れ Step5・FAQ |
| 「平日20時まで・土日祝営業」 | FVバッジ・店舗情報・FAQ |
| 「肩こり訴求」（C1） | 共感パートの筆頭・対応症状リスト筆頭 |
| 「腰痛訴求」（C1） | 共感パート2番手・対応症状リスト2番手 |
| 「姿勢矯正訴求」（C3） | FVキャッチ「その肩こり、姿勢が原因かもしれません」と直結 |

### 出し分け運用（応用）

URL末尾にUTMパラメータを付与し、LP側のJSで以下のような最適化を行うと、メッセージマッチ精度がさらに上がります（実装フェーズで対応）：

```
https://example.com/lp?utm_source=google&utm_medium=cpc&utm_campaign=c1_katakori
　→ FVキャッチを「肩こりの方、姿勢から見直しませんか？」に動的差し替え

https://example.com/lp?utm_source=google&utm_medium=cpc&utm_campaign=c3_shisei
　→ FVキャッチを「姿勢矯正は、整体×AIで。」に動的差し替え
```

## 4-9. A/Bテストロードマップ

| 優先度 | テスト対象 | 仮説 |
|---|---|---|
| 1 | FVキャッチコピー（A vs B vs C） | 「姿勢訴求」「時短訴求」「継続価値訴求」のどれが最もCVR高いか |
| 2 | CTAボタン文言 | 「初回3,500円で予約」 vs 「LINEで30秒予約」 |
| 3 | FV画像（施術風景 vs AI分析画面 vs 院内） | 何が最も信頼感を醸成するか |
| 4 | 中間CTAの位置 | S4直下 vs S6直下 vs S7直下 |
| 5 | 料金セクション内の回数券訴求の強弱 | LTV vs 初回CVRのトレードオフ |
| 6 | 共感パートのチェックリスト数（5項目 vs 10項目） | 認知負荷とエンゲージメントのバランス |
