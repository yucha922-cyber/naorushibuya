# 03. 薬機法・景品表示法・ビフォーアフター対応

> 整体院（非医療施設）におけるLP上の表現について、薬機法・景品表示法・医療広告ガイドラインを踏まえた**使用可能な表現／NG表現／必須注釈／チェックリスト**をまとめます。
> 本ドキュメントの基準は、コピーライティング・ビフォーアフター掲載・お客様の声・広告文すべてに適用されます。

---

## 3-1. 関連法令と適用関係（前提整理）

### 整体院（NAORU整体）に関係する法令

| 法令 | 適用関係 | 概要 |
|---|---|---|
| **薬機法**（医薬品医療機器等法）<br>**第66条**（誇大広告等の禁止） | 直接的には医薬品・医療機器が対象。ただし「効能効果を標榜」する表現全般に間接的影響あり | 「医薬品的な効能効果」をうたうと違反リスク。整体は医療行為ではないため、医療的効果の標榜は禁止 |
| **景品表示法**<br>**第5条**（不当な表示の禁止） | **すべての広告に適用** | 優良誤認表示（実際より良く見せる）、有利誤認表示（実際より有利に見せる）が禁止。No.1表記、ビフォーアフター、体験談、打消し表示すべて対象 |
| **医療広告ガイドライン**（厚生労働省） | 直接的には医療機関が対象。**整体院は対象外**だが、運用上「準じた配慮」が求められ、トラブル時の参考基準となる | 体験談・ビフォーアフター・最上級表現の規制が含まれる |
| **柔道整復師法・あはき法**（あん摩マッサージ指圧師、はり師、きゅう師等に関する法律） | 国家資格者が行う施術が対象。**民間整体は直接対象外**だが、誤認させる広告は問題化リスク | 「治療」「医業類似行為」を想起させる表現は避ける |
| 消費者庁「**打消し表示**に関する実態調査報告書」（2017） | すべての広告に適用される実務指針 | 注釈（打消し表示）の有効性に関する基準。文字サイズ・配置・タイミングが不適切だと「打消し表示として機能しない」と判断される |

### 整体（NAORU）の立ち位置の整理

NAORU整体は**民間資格・国家資格スタッフによる整体サービス**であり、医療機関ではありません。
したがって：

- **可能**：身体のコンディション、姿勢、可動域、コリ、張り、リラクゼーションに関する表現
- **不可**：医療行為を想起させる表現（治療・診断・処方・改善・効果・治る・No.1治療実績 等）

## 3-2. NG表現リスト（絶対に使用しない）

### A. 効果効能の断定／医療連想

| NG表現 | 理由 | 代替表現 |
|---|---|---|
| ❌ 治る／治す／治療／治癒 | 医療行為と誤認、薬機法・景表法違反 | ◯ 「お身体を整える」「コンディションを整える」 |
| ❌ 完治／根治／完全に治る | 同上＋効果保証の問題 | ◯ 「変化を実感される方が多い」 |
| ❌ 効果がある／効きます | 薬機法（医薬品的効能効果）連想 | ◯ 「期待できる」「お声をいただいています」 |
| ❌ 改善する／改善される（断定形） | 効果保証、薬機法リスク | ◯ 「変化を感じられる方が多い」「軽さを感じる方も」 |
| ❌ 〇〇に効く／〇〇が消える | 効能効果の標榜 | ◯ 「〇〇でお悩みの方にご好評」 |
| ❌ 診察／診断／処方／問診（医療用語） | 医療行為連想 | ◯ 「カウンセリング」「ヒアリング」「お身体の確認」 |
| ❌ ヘルニア／坐骨神経痛／腰椎すべり症 など疾患名 | 医療類似行為と誤認させると問題 | ◯ 「腰の張り」「腰の重さ」「腰回りの不調」 |
| ❌ 医学的に／科学的根拠／論文で証明 | 客観的根拠の明示なしには使用不可 | ◯ 出典明記の上で「〇〇大学の研究によると…」程度に留める |
| ❌ 自律神経が整う／血流が良くなる | 医学的効果の標榜 | ◯ 「リラックスしていただける」 |
| ❌ ホルモンバランス／免疫力アップ | 医学的効果の標榜 | ◯ 使わない |

### B. 最上級・No.1表記

| NG表現 | 理由 | 代替表現 |
|---|---|---|
| ❌ 渋谷No.1／日本一／業界トップ | 客観的根拠なしの最上級表現は景表法（優良誤認） | ◯ 客観的調査で根拠がある場合のみ「〇〇調べ・〇〇年〇月期」と明記して使用可。**原則使わない方が安全** |
| ❌ 唯一の／他にはない | 同上 | ◯ 「当院ならではの」「私たちが大切にしている」 |
| ❌ 絶対／必ず／100% | 効果保証 | ◯ 「多くの方が」「お声として」 |
| ❌ 最高／最強／最も効果的 | 最上級表現＋効果断定 | ◯ 「ご納得いただける」「丁寧な」 |
| ❌ 業界初／日本初 | 客観的根拠なしの優位性表記 | ◯ 客観的根拠がある場合のみ、出典明記で慎重に |

### C. ビフォーアフター関連の禁止事項

| NG | 理由 |
|---|---|
| ❌ 症状改善のビフォーアフター（痛みのある表情→笑顔） | 効果保証、誇大広告 |
| ❌ 「〇回でこんなに変わった！」のような回数と結果の断定 | 効果保証 |
| ❌ 注釈なしのビフォーアフター | 景表法上の打消し表示義務違反リスク |
| ❌ 加工・修正された写真 | 不当表示（実際と異なる） |
| ❌ 異なる撮影条件（光・角度・服装）でのビフォーアフター比較 | 実際と異なる比較、優良誤認 |
| ❌ 短期間（1回・1週間など）での劇的変化を強調 | 誇大広告、誤認誘発 |

### D. 体験談関連の禁止事項

| NG | 理由 |
|---|---|
| ❌ 「〇〇が治った」「〇〇が完全に消えた」という体験談 | 効果効能の断定（本人の感想であっても） |
| ❌ 「ヘルニアが良くなった」「坐骨神経痛が改善」など疾患名と効果を結びつけた体験談 | 医療類似行為と誤認 |
| ❌ 創作（架空）の体験談 | 景表法（不実証広告） |
| ❌ サクラ・関係者による体験談 | ステマ規制（2023年10月施行） |
| ❌ 注釈なし／打消し表示なしの体験談 | 景表法 |
| ❌ 極端な事例のみを集めた体験談 | 平均的な結果を装う優良誤認 |

## 3-3. OK表現リスト（推奨される使用表現）

### A. 身体の状態に関する表現

| OK表現 | ニュアンス |
|---|---|
| ◯ 身体が軽くなった感じがする | 主観的な感想 |
| ◯ コリ・張りが和らぐ／緩む | 一般的な感覚表現 |
| ◯ リラックスしていただける | リラクゼーション |
| ◯ 可動域が広がる／動きがスムーズになる | 客観的な身体の動き |
| ◯ 姿勢が変わる／姿勢のバランスが整う | 姿勢の客観的状態 |
| ◯ 肩・首・腰の重さを感じにくくなる方が多い | 感想ベース |
| ◯ お身体を整える | 包括的な表現 |
| ◯ コンディションをサポートする | サポート的な表現 |

### B. 期待値や効果に言及するときの表現

| OK表現 | ニュアンス |
|---|---|
| ◯ 〜の方にご好評いただいています | お客様の声を根拠とする |
| ◯ 〜とおっしゃる方が多いです | 主観的感想の集積 |
| ◯ 〜の変化を実感される方も | 一部の方の体験 |
| ◯ 〜が期待できます | 効果の可能性 |
| ◯ 〜のサポートを目指します | 目標としての表現 |

### C. 強み・特徴を語るときの表現

| OK表現 | ニュアンス |
|---|---|
| ◯ 国家資格者が在籍 | 事実 |
| ◯ AI姿勢分析を導入 | 設備の事実 |
| ◯ ボキボキしない／強圧を行わない | 施術スタイルの事実 |
| ◯ 渋谷で多くのお客様にご利用いただいています | 控えめな実績訴求 |
| ◯ ご納得いただけるご説明を心がけています | 姿勢・スタンス |

## 3-4. ビフォーアフター掲載：実施可能な範囲と条件

### 整体院で使用可能なビフォーアフターの種類

| 種類 | 使用可否 | 条件 |
|---|---|---|
| **姿勢の写真（横向きシルエット）** | ◯ 使用可 | 客観的な姿勢の比較。注釈必須 |
| **AI姿勢分析の数値・グラフ** | ◯ 使用可 | データの推移として提示。注釈必須 |
| **可動域の写真（前屈・側屈など）** | ◯ 使用可 | 動きの変化として提示。注釈必須 |
| **症状改善のビフォーアフター（痛みのある表情→笑顔）** | ❌ 使用不可 | 効果保証となる |
| **体型の変化（痩身など）** | △ 慎重に | 整体は痩身を目的としないため、原則扱わない |
| **顔の表情の変化（明るくなった等）** | ❌ 使用不可 | 効果保証となる |

### 必須注釈文言（テンプレート）

**全ビフォーアフター掲載時に、写真の直下に必ず表示**：

```
※掲載写真はお客様ご本人の同意のもと、施術前後の姿勢を撮影したものです。
　施術の結果には個人差があり、本写真は同様の変化を保証するものではありません。
　また、本写真は症状の改善・治癒を示すものではありません。
```

**注釈の表示要件（景表法・打消し表示ガイドライン準拠）**：
- 文字サイズ：本文と同等以上（最低14px、推奨16px）
- 配置：写真の直下、視線移動が自然な位置
- 色：背景と十分なコントラスト（グレー文字は不可）
- 動画の場合：表示時間は最低3秒、画面占有率は十分に
- SP表示時もフォントサイズは10px以下にしない

### ビフォーアフター掲載チェックリスト

実装前に以下をすべてYESにする：

- [ ] 撮影条件（光・角度・服装・距離・姿勢）はビフォーとアフターで統一されているか
- [ ] 写真の加工・修整は行っていないか
- [ ] 顔出しは行っていないか（行う場合は同意書を取得し、目隠し・モザイクを推奨）
- [ ] お客様本人から書面で掲載同意を取得しているか
- [ ] 必須注釈が写真の直下に明記されているか
- [ ] 注釈の文字サイズは本文と同等以上か
- [ ] 「治った」「効果」「改善」等のNG表現を含んでいないか
- [ ] 疾患名（ヘルニア等）と組み合わせていないか
- [ ] 短期間での劇的変化を強調していないか
- [ ] 「〇回で変化」など回数と結果を保証していないか
- [ ] 極端な事例のみを集めていないか（平均的な事例を含めているか）
- [ ] AI姿勢分析の数値を併記する場合、改ざんしていないか

## 3-5. お客様の声掲載：実施可能な範囲と条件

### 掲載してよい内容

- 主観的な感想（「身体が軽く感じた」「気持ちよかった」「スタッフが丁寧だった」）
- 客観的な姿勢の変化に関する感想（「姿勢が変わった気がする」「写真で違いが分かった」）
- 施術の感想（「ボキボキしないので安心だった」「説明が丁寧だった」）
- 通いやすさに関する感想（「駅から近い」「予約が取りやすい」「営業時間が良い」）
- AI姿勢分析に関する感想（「数値で見えるのが分かりやすい」）

### 掲載してはいけない内容

- 「治った」「治癒した」「完治した」
- 「効果があった」「改善した」（断定形）
- 「ヘルニアが良くなった」「坐骨神経痛が消えた」（疾患名＋効果）
- 「〇回で完全に変わった」（回数と結果の保証）
- 「他院では治らなかったのに、ここで治った」（他院との優劣比較で効果断定）
- 医学的・薬学的な効果に言及するもの

### 必須注釈文言（テンプレート）

**お客様の声セクションの冒頭または各カードに、必ず明示**：

```
※掲載されているご感想は、お客様個人の主観によるものであり、
　施術効果には個人差があります。同様のご感想・結果を保証するものではありません。
```

### ステマ規制対応（2023年10月施行・景表法）

- 関係者（スタッフ・家族・取引先）が一般顧客を装って書き込むことは禁止
- 報酬を渡して書いてもらった声を「広告」と明示せずに掲載するのは禁止
- インフルエンサー紹介の場合も「PR」「タイアップ」等の明示が必要
- 自社による感想であっても「お客様の声」と称するのは禁止

## 3-7. LP完成前 最終コンプライアンスチェックリスト

LP公開前に、以下をすべて確認：

### 文言チェック
- [ ] 「治る」「治療」「治癒」「完治」「効果」「効く」「改善」（断定形）が含まれていない
- [ ] 疾患名（ヘルニア・坐骨神経痛・椎間板等）が含まれていない
- [ ] 「No.1」「日本一」「最高」「絶対」「100%」等の最上級・絶対表現が含まれていない（または客観的根拠を明示している）
- [ ] 「医学的に」「科学的に証明」等の医療的根拠を示唆する表現が含まれていない
- [ ] 「自律神経」「ホルモン」「免疫力」等の医学的効果表現が含まれていない

### ビフォーアフターチェック
- [ ] ビフォーアフターは「姿勢」「可動域」「AI数値」に限定している
- [ ] 各ビフォーアフターに必須注釈が明記されている
- [ ] 注釈の文字サイズが本文と同等以上である
- [ ] 撮影条件が統一されている
- [ ] お客様の掲載同意書を取得している

### お客様の声チェック
- [ ] 体験談は主観的感想に限定されている
- [ ] セクション冒頭または各カードに必須注釈が明記されている
- [ ] 創作・サクラの体験談が含まれていない
- [ ] 関係者による体験談が含まれていない（含まれる場合はPR表示）
- [ ] 極端な事例のみを集めていない

### 比較表チェック
- [ ] 特定の競合名を明記していない
- [ ] 「一般的な〇〇」等の表現を使用している
- [ ] 客観的事実に基づく比較である

### 注釈・打消し表示チェック
- [ ] 必須注釈の文字サイズが本文と同等以上である
- [ ] SP表示時も注釈が読める
- [ ] 注釈の色と背景にコントラストがある（薄いグレー文字は不可）

### 法令確認
- [ ] 景品表示法（不当表示）に該当する表現がない
- [ ] 薬機法（誇大広告等）に該当する表現がない
- [ ] ステマ規制に該当する内容がない
- [ ] 医療広告ガイドラインに準じた表現になっている

### 推奨：第三者チェック
- [ ] 薬機法・景品表示法に詳しい専門家（弁護士・行政書士・薬機法研究所等）に最終チェックを依頼
