# スタッフ紹介セクション 写真

このディレクトリには、「あなたの身体を担当する、国家資格者をご紹介。」セクションの**施術者の写真**を配置します。

## 画像の配置方法

以下のファイル名で画像をアップロードすると、自動でカードに表示されます。HTML/CSSの編集は不要です。

| 施術者 | ファイル名 |
|---|---|
| 院長 洪 侑沙 | `hong-yusa.jpg` |
| 施術者 酒井 翔子 | `sakai-shoko.jpg` |

画像が未配置・読み込み失敗の場合は、自動でプレースホルダー（`placeholder-hong.svg` / `placeholder-sakai.svg`）が表示されます。

## 推奨画像仕様

| 項目 | 推奨 |
|---|---|
| 形式 | JPG / WebP |
| アスペクト比 | **3:4（縦長・ポートレート）** ※カードは3:4で固定表示 |
| 推奨サイズ | 600×800px 以上 |
| ファイルサイズ | 300KB以下推奨 |

- 写真はカード上部に `object-fit: cover`（中央上寄せ）で表示されます。顔が中央〜上に来る縦構図がきれいに収まります。
- カードは縦横比を固定しているため、画像差し替え時にレイアウトがずれません（CLS対策済み）。

## 別のファイル名を使いたい場合

`index.html` のスタッフセクション内の `<img src="...">` を編集してください。
