# 比較（Comparison）セクション 背景画像

このディレクトリには、「他の整体と、ここが違います。」セクションの**背景画像**を配置します。

## 画像の配置方法

1. このディレクトリに **`bg.jpg`** というファイル名で画像をアップロードしてください
2. アップロードと同時に、自動でセクションの背景に表示されます
3. HTMLやCSSを編集する必要はありません

画像が未配置・読み込み失敗の場合は `placeholder.svg` が自動表示されます。

## 推奨画像仕様

| 項目 | 推奨 |
|---|---|
| ファイル名 | `bg.jpg` |
| 形式 | JPG / WebP |
| アスペクト比 | 16:9 推奨（横長画像） |
| 推奨幅 | 1600〜1920px |
| ファイルサイズ | 400KB以下推奨 |
| 構図イメージ | 院内・施術空間・落ち着いた雰囲気 |

## 仕様

- 画像の上に**明るい半透明オーバーレイ（約88〜92%）**が自動で重なり、比較表の濃い文字が読みやすく保たれます
- 比較表自体は白背景のため、背景画像の上でも常に視認可能です
- もっと画像を見せたい / 暗いトーンにしたい場合は `assets/css/main.css` の `.compare__overlay` の不透明度・色を調整してください

## 別のファイル名・形式を使いたい場合

`index.html` 内の以下を編集してください：

```html
<img
  src="assets/img/compare/bg.jpg"  ← ここを変更
  ...
  onerror="this.onerror=null;this.src='assets/img/compare/placeholder.svg';">
```
