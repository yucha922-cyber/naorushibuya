# ビフォーアフターセクション 画像（6枚）

このディレクトリには、「姿勢の変化を、ご覧ください。」セクションの**ビフォーアフター写真（3事例×2枚）**を配置します。

## 画像の配置方法

各事例の施術前（BEFORE）・施術後（AFTER）の写真を、以下のファイル名でアップロードしてください。

| 事例 | BEFORE | AFTER |
|---|---|---|
| 事例1（30代男性／IT企業） | `case1-before.jpg` | `case1-after.jpg` |
| 事例2（40代女性／広告代理店） | `case2-before.jpg` | `case2-after.jpg` |
| 事例3（50代男性／経営者） | `case3-before.jpg` | `case3-after.jpg` |

### GitHub Web画面からアップロードする手順

1. GitHub のリポジトリページで `assets/img/beforeafter/` ディレクトリを開く
2. 右上の「Add file」→「Upload files」をクリック
3. 画像ファイルをドラッグ＆ドロップ
4. **ファイル名が上記の通りであることを確認**
5. 「Commit changes」をクリック

## 推奨画像仕様

| 項目 | 推奨 |
|---|---|
| 形式 | JPG / WebP |
| アスペクト比 | **3:4 推奨**（縦長・600×800px 等） |
| ファイルサイズ | 各 **150KB以下推奨** |
| 撮影 | BEFORE・AFTERは**同じ条件**（光・角度・服装・距離）で撮影 |

## ⚠️ 薬機法・景品表示法に関する重要な注意

ビフォーアフター写真には法的な制約があります。必ず以下を守ってください（詳細は `docs/03-compliance-and-beforeafter.md` 参照）。

- **姿勢の客観的な変化のみ**を扱う（横向きシルエット推奨）
- 「治った」「改善した」など**症状改善の表現はNG**
- 顔は写さない or ぼかす（プライバシー保護）
- お客様本人から**書面で掲載同意**を取得する
- 撮影条件（光・角度・服装）を BEFORE/AFTER で統一する
- セクション下部の必須注釈（個人差がある旨）は削除しない

## 仕様

- 各画像はカード内に `object-fit: cover` で表示され、左上に BEFORE / AFTER バッジが重なります
- 画像が未配置の場合は `placeholder.svg`（横向き姿勢シルエット）が自動表示されます
